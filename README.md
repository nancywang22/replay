# replay

replay is a full-stack web application that converts real concert setlists into
Spotify playlists by integrating the Setlist.fm and Spotify APIs.

## Features
- Fetch official concert setlists
- Normalize and order songs
- Generate Spotify playlists from setlists
- Cache concerts and playlists to avoid duplicate API calls

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
